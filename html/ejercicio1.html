Quiz #2 Review

Cada cuestionario contendrá 25 puntos de actividades para completar en el ordenador. Por lo tanto, es importante que sea capaz de utilizar el software y las técnicas tratadas en las sesiones de laboratorio.

Excel parte del cuestionario: Se le pedirá que haga lo siguiente:

  • Deberías ser capaz de ordenar listas de Excel (varias formas de ordenar como se ha visto en clase), incluyendo la ordenacion por campos múltiples.
  • Deberia poder usar Autofiltros, crear sus propios Autofiltros personalizados y aplicar formatos condicionales.
  • Deberia poder mostrar los totales y crear subtotales para una lista.
  • Se le pedirá que CREE MÚLTIPLES TABLAS PIVOTES.
  • Deberia poder crear una tabal de datos de 1 y 2 variables (capitulo de analisis hipotetico capitulo: prepráctica #2.)

  • Revisar elementos
    1. Funciones de Excel (PAGO, SI, BUSCARV).
    2. Ser capaz de crear una formula que se refiera a datos (extrae datos) en otra hoja (nombres y números).
    3. En un entrono empresarial, saber cómo calcular el impuesto sobre las ventas.

XML

Podría darle una tabla de posibles etiquetas para usar (como vio en la primera parte del proyecto XML) y se le pedirá que use ese diagrama para crear un documento de instancia XML (elemento raíz, elementos de nivel medio, atributos, y el contenido real de los datos). El documento instancia XML resultante debe estar bien formado y debe ajustarse a la tabla de posibles etiquetas que se proporciona. NO OLVIDE que las etiquetas XML NO PUEDEN CONTENER ESPACIOS dentro de la etiqueta real < XMLCommand>.

Tambien podriá darle un archivo XML con errores y tendra que hacerlo bien formado. Asegúrese de revisar los requisitos de un documento de instancia bien formado. NO OLVIDE que las etiquetas XML NO PUEDEN CONTENER ESPACIOS dentro de la etiqueta real < XMLCommand>.
